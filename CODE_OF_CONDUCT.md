# Code of Conduct

ZeroDaySentinel is a defensive cybersecurity research project. Contributors and users are expected to communicate respectfully, act responsibly, and avoid any activity that could enable harm.

## Expected Behavior

- Be respectful and constructive.
- Keep discussions focused on defensive security.
- Avoid sharing harmful or weaponized content.
- Respect legal and ethical boundaries.
- Help maintain a safe learning environment.

## Unacceptable Behavior

- Harassment or abusive language.
- Sharing exploit code or harmful payloads.
- Encouraging unauthorized access.
- Publishing sensitive vulnerability details without coordination.
- Promoting malicious activity.

## Enforcement

Maintainers may remove content, close issues, reject pull requests, or restrict participation if behavior violates the purpose of this project.
