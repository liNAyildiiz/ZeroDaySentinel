ZeroDaySentinel is developed as a defensive zero-day readiness and detection engineering lab.

- Repository structure
- Ethical safety boundary
- Synthetic telemetry generator
- Defensive detector prototype
- CLI entry point
- Sigma-style detection example
- YARA-style marker rule
- Architecture documentation
- Demo output documentation
- Automated tests
- GitHub Actions CI validation
- Versioned releases

## Safety Boundary

ZeroDaySentinel will remain defensive, synthetic, and non-weaponized. The project will not include exploit code, payloads, bypass techniques, persistence logic, evasion logic, unauthorized testing instructions, or internet-wide scanning tooling.
